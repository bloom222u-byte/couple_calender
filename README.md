# OurDays v1.5

## 수정 내용

- 모바일 달력 칸에서 일정/공휴일/절기 앞 아이콘을 제거했습니다.
- 색상 배경만으로 남친/여친/데이트/공휴일/절기를 구분합니다.
- 모바일에서 일정 제목 앞 2~3글자라도 보이도록 태그 여백과 글자 크기를 조정했습니다.
- 사진이 있는 날짜는 기존처럼 사진만 밝게 표시됩니다.
